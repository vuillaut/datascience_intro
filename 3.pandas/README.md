## Update your clone before starting

```
git pull
```

If you can't rebase (because of local changes in other parts of the lecture):

1. Stash current work (including untracked files)
```
git stash push -u -m "wip before pull"
```

2. Pull with rebase (avoid useless merge commits)
```
git pull --rebase
```

3. Re-apply your stash
```
git stash pop
```
